CompuMas - "Info"

🧪 CompuMas: Branch Flow.

Este proyecto usa un sistema de ramas escalonado para desarrollo seguro y modular.

🔀 Ramas activas

  • stash: Caja de – ideas, prototipos y pulgas.
  • devops: Ambiente de desarrollo controlado. Aquí se testean módulos individuales.
  • factory: Ensamble de módulos. Aquí se prueba integración total.
  • main: Versión estable y final. Solo entran cambios pulidos.

🔁 Flujo recomendado.

stash → devops → factory → main

🧠 Tips

  • stash: No vivas ahí. Solo entra y sal rápido.
  • devops: Ideal para experimentar con herramientas, UI, lógica individual.
  • factory: Tu zona de pruebas bravas antes de pasar a producción.
  • main: Intocable. Lo que está aquí, debe funcionar sí o sí.

🧠 Herramientas Utiles

  • WinDirStat: Analisis de directorios
  • RustDesk: Remote Desktop Ultralite.
  • Cobian Backup: Automatizacion de Backups y Tareas programadsa de mantenimiento.
  • Kasperky Cybersecurity: Antivirus y productos y servicios avanzados de ciberseguridad.
  • WinRar: Potente Compresor.
